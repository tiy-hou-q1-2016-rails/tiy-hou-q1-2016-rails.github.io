Install rbenv & ruby-build
--------------------------

> brew install ruby-build rbenv  
> echo 'if which rbenv > /dev/null; then eval "$(rbenv init -)"; fi' >> ~/.profile

Install ruby 2.2.2
------------------

1.	Close and reopen terminal.
2.	> rbenv install 2.2.2
3.	Stretch
4.	> rbenv global 2.2.2
